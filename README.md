# Emoji Replacer

文章の句読点や改行を絵文字に置き換えて、ページの表示を親しみやすく（おふざけ）変換するChrome拡張機能です。

## 機能
- 「。」を絵文字（🤗 / 💗）に交互変換
- 「<br>」を絵文字＋改行に交互変換
- 「、」を絵文字に変換

## インストール
1. `chrome://extensions` を開く
2. 右上「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」→ このフォルダを選択

## ファイル構成
- `manifest.json` : 拡張機能の設定
- `main.js` : 変換処理本体
- `content.js` : ページへの注入スクリプト

## 注意
`innerHTML`を書き換えるため、サイトによっては表示が崩れる場合があります。
